# Workout Tracker

A full-stack workout tracking web application that allows users to record exercises, manage workout history, and visualize progress through charts.

## Features

- Add workouts
- Edit workouts
- Delete workouts
- View workout history
- Workout progress chart

## Tech Stack

Frontend:
- React (Vite)

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Project Structure

workout-tracker
│
├── frontend      # React frontend
├── server        # Node.js backend
└── README.md

## How to Run Locally

1. Clone the repository

git clone https://github.com/Aryann0907/workout-tracker.git

2. Start backend

cd server  
npm install  
node server.js  

3. Start frontend

cd ../frontend  
npm install  
npm run dev
